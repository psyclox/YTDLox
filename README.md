# YTDLox

**YTDLox** is a modern, premium desktop web interface for downloading videos and audio. It is a fork and spiritual successor to [YTDLnis](https://github.com/deniscerri/ytdlnis) and other YTDL projects, bringing their powerful functionality into a clean, highly configurable, and user-friendly web UI.

## Features
- **Max Quality Control**: Easily select the highest quality video and audio streams (e.g., 1080p60, 4K, MP3 320kbps).
- **Format Flexibility**: Switch easily between MP4, MKV, WebM, and MP3 containers.
- **Advanced Adjustments**: Toggle features like Subtitles, SponsorBlock, Thumbnail embedding, and Chapters directly from the interface.
- **Dynamic Command Generation**: Automatically builds the corresponding `yt-dlp` or standard download command based on your UI selections.
- **Stable Transmission**: Engineered for seamless functionality and stable downloads wrapped in a premium dark-mode, glassmorphic design.

## How to Use
1. Clone or open the project directory.
2. Open `index.html` in your favorite modern desktop browser.
3. Paste the video URL in the bottom input bar.
4. Select your preferred output under the Video or Audio tabs.
5. Click the Download icon on the bottom right.
