# YTDLox

**YTDLox** is a modern, premium desktop interface for downloading videos and audio. It is a fork and spiritual successor to [YTDLnis](https://github.com/deniscerri/ytdlnis) and other YTDL projects, bringing their powerful functionality into a clean, highly configurable, and user-friendly desktop UI.

## Features
- **Max Quality Control**: Easily select the highest quality video and audio streams (e.g., 1080p60, 4K, MP3 320kbps).
- **Format Flexibility**: Switch easily between MP4, MKV, WebM, and MP3 containers.
- **Advanced Adjustments**: Toggle features like Subtitles, SponsorBlock, Thumbnail embedding, and Chapters directly from the interface.
- **Multiple Audio Streams**: YTDLox specializes in MKV containers, allowing you to fetch subtitles and package them with multiple audio formats. When multiple audio streams are available, you can choose exactly which ones to include, defaulting to the highest quality automatically.
- **Playlist & Search Support**: Built-in Youtube search engine and full playlist parsing.
- **Stable Transmission**: Engineered for seamless functionality and stable downloads wrapped in a premium dark-mode, glassmorphic design.

## User Instructions
1. Run the standalone `.exe` Application. No Node.js server or `npm` install is required to run YTDLox natively!
2. In the Left Sidebar, you can search for a video directly or paste a URL/Playlist link in the bottom-center bar.
3. The app will automatically fetch the video metadata, duration, and highest quality formats!
4. **Settings:** Click the Settings icon in the top right to configure your Default Download Directory and Max Concurrent Downloads.
5. **Downloading:** Select your preferred output under the Video or Audio tabs and click the Download icon.
6. **Audio Selection Modals:** If a video format lacks built-in audio and multiple audio streams exist, the app will prompt you to select one or multiple audio tracks to mux into the final MKV.

> **💡 Tip for MKV Files:** If you downloaded an MKV file containing multiple audio tracks or subtitles, and VLC player shows a white screen or struggles to play it, we highly recommend using **[PotPlayer](https://potplayer.tv/)**. It offers superior out-of-the-box support for complex MKV containers.
